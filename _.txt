[ 컴파일 ] : 컴퓨터가 이해할 수 있는 언어로 번역하는 프로그램, 컴파일 : 번역하는 작업

[ 자바 장점 ]
    1. 운영체제(플랫폼)으로부터 독립적이다. 코드(.class)를 한번 작성하면 여러 운영체제에서 실행할 수 있다.
    2. 객체지향언어(상호관계), 유지보수, 확장성이 좋다.
    3. 가비지 컬렉터 (GC : ) : 메모리 자동으로 관리
    4. 오픈소스(JDK, SPRING, JSP) : 남이 만들어둔 소스를 공개적으로 사용가능하다.

[ 자바 사용처 ] : 웹, 앱, 게임

[ JDK 설치 확인 ]
    1. cmd 열기
    2. java -version

[ 이클립스 설치 ] 중요하지 않은 부분
    IDE(통합개발환경) : 코드를 작성하는데 여러가지 기능 제공하는 프로그램
        - 1. VSCODE : (무료)프론트엔드 개발자에서 주로 사용
        - 2. 이클립스 : (무료)백엔드 개발자에서 주로 사용
        - 3. 인텔리제이 : (부분유료) 프/백
